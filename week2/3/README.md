# 직사각형 둘레 및 넓이

## 설명
분수 형태의 가로, 세로를 외부 인자로 입력받아 둘레와 넓이를 출력하는 프로그램

## 사용법
윈도우: .\program.exe 가로 세로
맥:     ./program 가로 세로

## 입력
- 가로, 세로는 A/B 형태의 분수, 정수, 소수 형태로 입력 가능

## 출력
- Perimeter: 둘레 (기약분수)
- Area: 넓이 (기약분수)

## 예시
입력: .\program.exe 3/7 4/9
출력:
Perimeter: 110/63
Area: 4/21

## 계산 공식
- 둘레 = 2 * (가로 + 세로)
- 넓이 = 가로 * 세로

## 오류 처리
- 인자 개수가 2개가 아닌 경우 오류 메시지 출력
- 분모가 0인 경우 오류 메시지 출력
- 잘못된 입력 형식인 경우 오류 메시지 출력
