## 분수 ADT (Fraction)

### 자료형
- numerator: 정수형 분자
- denominator: 정수형 분모

### 연산
- reduce(f): 분수 f를 기약분수로 변환하여 반환
- add_frac(a, b): 두 분수 a, b를 더한 기약분수 반환
- sub_frac(a, b): 두 분수 a, b를 뺀 기약분수 반환
- mul_frac(a, b): 두 분수 a, b를 곱한 기약분수 반환
- div_frac(a, b): 두 분수 a, b를 나눈 기약분수 반환
- str_to_frac(str): "A/B" 문자열을 분수 자료형으로 변환
- print_frac(f): 분수를 A/B 형태로 출력 (분모가 1이면 정수 출력)
