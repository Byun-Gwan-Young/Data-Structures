# GCD, LCM

## GCD 알고리즘 (유클리드 호제법)
1. 두 정수 a, b를 입력받는다
2. b가 0이 될 때까지 반복한다
   - t = b
   - b = a % b
   - a = t
3. a를 반환한다

예시: GCD(18, 24)
a=18, b=24
t=24, b=18%24=18, a=24
t=18, b=24%18=6,  a=18
t=6,  b=18%6=0,   a=6
b=0 → return 6

## LCM 알고리즘
1. 두 정수 a, b를 입력받는다
2. LCM = |a| / GCD(a, b) * |b|
3. 반환한다

예시: LCM(18, 24)
GCD(18, 24) = 6
LCM = 18 / 6 * 24 = 72
