과제(4) - 프로파일링

과제 제출 관련 안내 공지사항을 꼭 확인하세요.



1) GCD 복잡도

- 이전 과제에서 제출한 GCD 알고리즘에 대해 시간복잡도와 Big-O를 계산하고, Markdown(*.md) 파일로 보고서를 제출한다.

- 제출했던 GCD 알고리즘의 파일 링크 역시 함께 제출한다.



2) 피보나치 복잡도

- 피보나치 수열의 n번째 숫자가 F(n)이라 할 때, n을 5부터 순서대로 증가시키며 GCD(F(n), F(n-1))을 계산하며 CPU 사용량과 메모리 사용량을 프로파일링한다. (피보나치 수열은 재귀적 방법으로 구현한다)

- 그 결과를 바탕으로 GCD의 시간복잡도 및 피보나치 수열의 시간복잡도를 검증하고, 해당 2) 과제의 코드 및 Big-O 계산식과 프로파일링 한 그래프 이미지 등을 포함하여 역시 Markdown(*.md) 파일로 보고서를 제출한다.



- VS를 사용할 수 없는 경우, time.h의 clock() 함수를 이용하여, 프로그램의 시작과 끝의 시간 차이를 구하고, 이를 출력하여 프로파일링 그래프 이미지를 대신합니다.


+ 참고 사항

- 과제 제출 방법 세부 안내에서 0점 처리로 안내된 조항 및 잘못된/늦은 과제 제출의 정정은 불가합니다.
